# AI Code Review System 🔍

## 📌 Project Overview

The AI Code Review System is a full-stack web application that allows users to submit JavaScript code and receive AI-generated code review feedback. The project uses a backend API service and an interactive frontend editor interface.

## 🧠 Features

- Code submission using online editor
- AI-powered code review analysis
- Real-time review response display
- Syntax highlighting code editor
- Markdown formatted AI response rendering

## 🛠 Technologies Used

### Backend
- Node.js
- Express.js
- CORS middleware
- REST API architecture

### Frontend
- React.js
- Axios for API communication
- react-simple-code-editor
- PrismJS syntax highlighting
- React Markdown renderer
